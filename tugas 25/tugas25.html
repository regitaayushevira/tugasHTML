Mengenal Semantic Element Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer.
    karena article memiliki type block, artinya element ini memiliki lebar sesuai browseer jika kita tidak menentukan lebarnya.

  • aside

    • Element ini bisa di gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.
    karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak mennetukan lebarnya
